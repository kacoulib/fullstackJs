# Calculatrice avec bouton

Vous allez faire un petit jeu de Yam (5 dés).

Créez un bouton pour lancer le jeu de Yam. Affichez le résultat sous le bouton

A chaque que vous jouez garder le résultat dans un tableau. Puis créez les boutons respectifs suivants :

1. Un bouton pour calculer le nombre de Yam.

2. Un bouton pour calculer le nombre de carré.(faculatif)

3. Un bouton pour calculer le nombre de Brelan. (faculatif)

4. Un bouton pour reset pour vider le tableau. (faculatif)

```text
[Lancer]

 1 . 2 . 3 .  3  . 3 

[Yam]
[Carré]
[Brelan]

resultat : 

[Reset]
```
